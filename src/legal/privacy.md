## 引言

星汇盛世（北京）科技有限公司（以下简称「**我们**」或「**本公司**」）运营星汇小蜜书（MeetEasy）平台（以下简称「**本平台**」或「**星汇小蜜书**」）。我们重视您的个人信息保护，并依据《中华人民共和国个人信息保护法》（PIPL）、《网络安全法》《数据安全法》等法律法规制定本隐私政策。

本政策说明我们如何在您使用 Admin、Console、MeetApp、VisuSpace、微信小程序及相关 API 时收集、使用、存储、共享与保护个人信息，以及您享有的权利。

**请在使用服务前仔细阅读本政策。注册、登录或使用即表示您理解并同意本政策；若不同意，请停止使用。**

## 一、适用范围

1. 本政策适用于由本公司直接向您提供的星汇小蜜书 SaaS 服务。
2. 当您作为 **参会者** 向 **会务组织者**（租户）提交报名信息时，组织者可能作为独立的个人信息处理者；我们作为 **技术受托方/平台运营者** 按组织者指令与平台配置处理数据。组织者对收集目的负有首要说明义务，我们提供工具与安全保障。
3. 本政策不适用于第三方网站、插件 Sidecar 服务或组织者自行嵌入的外链内容；请向相应第三方查阅其隐私政策。

## 三、我们收集的个人信息

我们遵循 **合法、正当、必要和诚信** 原则，仅收集实现功能所必需的信息。

### 3.1 账号与身份

- 手机号码、电子邮箱
- 登录密码（存储为哈希值，非明文）
- 微信 OpenID、UnionID（小程序/微信登录场景）
- 姓名、头像（您主动填写或授权）

### 3.2 会务与报名

- 组织者在 AMIS 表单中配置的字段（公司、职位、证件号等 **由组织者决定**）
- 报名状态、票码、签到记录、座位分配
- 您上传的文件（资质证明等，如表单启用）

### 3.3 设备与日志

- IP 地址、浏览器/操作系统类型、设备标识符
- 访问时间、请求 URL、Referrer
- 崩溃日志与性能数据（脱敏后）

### 3.4 分析与营销归因

- 页面浏览（PV/UV）、分享渠道代码（`_s` 参数）
- 转化漏斗事件（匿名或假名化聚合）

### 3.5 AI 与语音

- 您与 AI 助手的 **对话内容**
- 语音输入经 ASR 转换的 **文本**
- 为生成回答而检索的会议上下文（议程、FAQ 等）

### 3.6 音视频

- 若会议启用直播、录播或 ASR，可能处理 **音频流**；具体以会议页面提示为准。

### 3.7 Cookie 与本地存储

- 会话 Token、Refresh Token
- 语言、主题偏好
- 分析会话 ID、分享归因 ID

您可通过浏览器设置管理 Cookie；禁用可能影响登录与部分功能。

## 五、我们如何使用个人信息

1. 提供、维护与改进星汇小蜜书功能。
2. 向组织者展示其租户范围内的报名与签到数据。
3. 发送服务通知（审核结果、会议提醒），您可退订营销类消息。
4. 调用 LLM/ASR **委托处理** 您的对话与语音（见第六节）。
5. 遵守法律法规、执法或监管要求。

我们不会将个人信息用于本政策未载明的目的；变更用途时将重新取得同意（法律另有规定除外）。

## 六、共享、委托处理与公开披露

### 6.1 共享

除以下情形外，我们不会向第三方出售您的个人信息：

- **会务组织者**：您向某会议提交的信息对该租户组织者可见。
- **法定情形**：依据法律、诉讼、政府要求。
- **经您同意** 的其他共享。

### 6.2 委托处理（子处理者）

我们可能委托以下类型服务商处理数据（具体名单随部署更新）：

| 类别 | 用途 |
|------|------|
| 云基础设施 | 主机、MongoDB、Redis、对象存储 |
| 微信开放平台 | 小程序登录、消息 |
| LLM / ASR 提供商 | AI 对话、语音识别 |
| 邮件/短信网关 | 通知发送 |

我们与子处理者签署数据处理协议，要求其采取不低于本政策标准的保护措施。

### 6.3 公开披露

仅在取得您明确同意或法律强制要求时公开披露。

## 八、存储期限

| 数据类型 | 保留期限（原则） |
|----------|------------------|
| 账户信息 | 账号存续期间；注销后 **30–90 日** 内删除或匿名化 |
| 报名与签到 | 会议结束后 **不少于 6 个月**（组织者导出需求）；最长不超过 **3 年** 或法律要求 |
| 日志 | **6–12 个月** |
| AI 对话 | **90 日** 或租户配置更短周期 |
| 分析聚合 | 长期保留匿名统计数据 |

组织者可在 Console 导出后申请删除租户数据（受法律留存限制约束）。

## 九、您的权利

依据 PIPL，您享有：

1. **查阅、复制** 您的个人信息
2. **更正、补充** 不准确的信息
3. **删除**（法定或约定情形）
4. **撤回同意**（不影响撤回前处理的合法性）
5. **注销账号**
6. **解释说明** 处理规则

**行使方式**：发送邮件至 [待补充]，或在 MeetApp/Console「账号设置」提交申请。我们将在 **15 个工作日** 内响应（复杂情形可延长并说明）。

## 十、未成年人保护

本平台主要面向成年人。若您未满 **14 周岁**，请在 **监护人同意** 后使用；我们不会主动面向儿童营销。发现未经监护人同意收集未成年人信息的，将尽快删除。
