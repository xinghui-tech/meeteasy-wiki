# 核心功能

星汇小蜜书围绕会议全生命周期设计，以下为核心能力模块。

## 会议生命周期

- **会议创建与发布**：支持多会议并行，配置基本信息、时间地点与可见性。
- **议程与会场**：多轨道议程、嘉宾资料、论坛分组，与会务微站联动展示。
- **内容交付**：附件（PDF、Office、Markdown）与知识库集成，便于资料分发。

## 灵活报名

- **低代码表单**：基于 AMIS 的表单设计器，自定义字段、校验规则与布局。
- **审核工作流**：支持人工审核、批量通过/驳回，状态可追溯。
- **票码与签到**：自动生成参会票码，与签到、座位模块联动。

## 互动体验

- **AI 助手**：集成 LLM（如 DeepSeek、OpenAI 等），为参会者提供智能问答。
- **语音输入**：ASR 语音识别，支持语音转文字与交互场景。
- **实时通信**：Socket.IO 驱动的即时消息与通知推送。

## VisuSpace 微站

- **可视化页面编辑**：低代码 DSL 描述页面结构与样式，AI 辅助生成。
- **组件化布局**：文本、图片、按钮、议程列表等业务组件开箱即用。
- **多端渲染**：微站页面在 MeetApp 中统一呈现。

## 数据分析

- **访问统计**：实时 PV/UV、页面路径与访客行为分析。
- **分享归因**：追踪分享链接与转化路径，辅助营销决策。
- **每日汇总**：自动化报表，供组织者与平台 Admin 查看。

## 插件扩展

- **内置插件**：AI、OCR、报名等预集成模块。
- **Embedded 插件**：Python 插件，与后端事件总线深度集成。
- **Sidecar 插件**：HTTP 服务插件，适合独立部署的扩展能力。

## 多租户与安全

- 租户级数据隔离、RBAC 权限控制。
- HTTPS 传输、会话 Token 与会话管理。
- 详见 [数据安全](/legal/data-security)。
