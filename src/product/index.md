# 产品概览

**星汇小蜜书（MeetEasy）** 是由 **星汇盛世（北京）科技有限公司** 运营的企业级会议管理 SaaS 平台，面向会务组织者、参会者与平台运营方，提供从会议创建、报名审核、现场签到到数据分析的全流程数字化能力。

## 我们解决什么问题

传统会务组织依赖 Excel、邮件与分散工具，常见问题包括：

- 报名信息分散，审核与票码管理效率低
- 议程、嘉宾、微站内容更新不同步
- 现场签到与座位查询体验差
- 传播效果难以量化，缺乏数据闭环

星汇小蜜书将上述环节整合为统一平台，通过多租户隔离与插件扩展，适配大型峰会、行业论坛、企业内训等多种场景。

## 目标用户

| 角色 | 使用端 | 典型任务 |
|------|--------|----------|
| 平台运营 | Admin | 租户管理、插件配置、全局模板与系统设置 |
| 会务组织者 | Console / WeConsole | 创建会议、配置报名、管理议程与签到 |
| 参会者 | MeetApp / WeApp | 浏览微站、报名、AI 问答、现场签到 |
| 内容设计 | VisuSpace | 可视化编辑会议微站页面 |
| 开发者 | API / 插件 | 二次开发、集成与运维部署 |

## 核心价值

1. **租户优先**：数据与配置按租户严格隔离，适合 SaaS 多客户运营。
2. **插件驱动**：AI、OCR、报名等能力以插件形式接入，可按需启用。
3. **体验一致**：MeetApp、Console 与文档站共享品牌视觉（主色 `#FF6A00`）。
4. **开放扩展**：Monorepo 架构与 webapi SDK，降低前后端协作成本。
