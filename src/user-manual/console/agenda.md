# 议程、嘉宾与论坛管理

Console 提供结构化的议程与嘉宾数据管理，数据与 VisuSpace 微站、MeetApp 列表组件联动。

## 议程管理

### 添加议程条目

1. 会议详情 → **议程** → **添加条目**。
2. 填写：
   - 开始/结束时间
   - 标题与副标题
   - 所属轨道/论坛（多轨道会议）
   - 关联嘉宾（从嘉宾库选择）
   - 会场/房间
3. 支持拖拽排序或按时间自动排序。

### 多轨道与论坛

大型会议可划分 **主论坛**、**分论坛** 等轨道：

1. **论坛管理** → 新建论坛（名称、描述、负责人）。
2. 创建议程条目时选择所属论坛。
3. MeetApp 可按论坛筛选议程。

## 嘉宾管理

### 嘉宾资料

| 字段 | 说明 |
|------|------|
| 姓名 | 展示名称 |
| 头像 | 建议 1:1，≥ 200×200px |
| 职务/公司 | 单行或分行展示 |
| 简介 | 支持富文本或 Markdown |
| 社交链接 | 可选 |

1. **嘉宾** → **添加嘉宾** → 填写资料 → 保存。
2. 在议程条目中 **关联嘉宾**，微站「嘉宾墙」组件可自动拉取列表。

## 与微站联动

VisuSpace 中可使用：

- **议程列表组件**：按日期/论坛过滤展示
- **嘉宾卡片组件**：网格或轮播展示
- **嘉宾详情页**：点击跳转单页

修改 Console 议程/嘉宾后，**重新发布微站** 或依赖动态数据源（视组件配置）更新展示。

## 导入导出

- 支持 CSV/Excel 批量导入议程（视版本功能）。
- 导出议程供印刷物或第三方日程工具使用。

## 最佳实践

- 议程时间在发布前与现场执行表对齐，避免 MeetApp 与手册不一致。
- 嘉宾头像统一裁剪比例，提升微站视觉质量。
- 分论坛负责人仅授予该论坛编辑权限（RBAC）。
