# 数据分析与分享追踪

Console 数据分析帮助组织者了解微站传播效果、访问趋势与转化情况。

## 分析看板

1. 会议详情 → **数据分析**。
2. 默认展示：
   - **PV / UV**（页面浏览量 / 独立访客）
   - **访问趋势**（按日/小时）
   - **热门页面**（VisuSpace 各页路径）
   - **来源渠道**（直接访问、分享、外链）

## 分享追踪

### 生成追踪链接

1. **分享管理** → **创建分享链接**。
2. 填写渠道名称（如「官方公众号」「嘉宾 A 朋友圈」）。
3. 系统生成带 **追踪参数** 的 MeetApp URL 或短链。
4. 分发后可在报表中查看各渠道 UV、报名转化数。

### 归因逻辑

- 访客首次通过追踪链接进入时，记录 **归因渠道**（Cookie / localStorage，见隐私政策说明）。
- 后续报名可关联至该渠道，用于评估 ROI。
- 归因窗口期可在平台配置中设定（默认 7–30 天，视部署而定）。

## 每日汇总

- 系统异步聚合前一日数据（FastStream Worker），Console 展示 **日报** 卡片。
- Admin 平台级可查看跨租户汇总（需权限）。

技术实现见 [分析与异步](/developer/analytics-async)。

## 导出与分享

- 支持导出 CSV/PDF 报表（视版本功能）。
- 只读分享链接可授予赞助商查看部分指标（不暴露个人报名信息）。

## 指标说明

| 指标 | 定义 |
|------|------|
| PV | 页面加载次数（含重复访问） |
| UV | 独立访客（按设备/会话去重） |
| 转化 | 完成报名且审核通过的人数 / UV |

## 隐私与合规

分析数据可能包含设备标识与行为日志，处理方式见 [隐私政策](/legal/privacy)。请勿将分析结果与可识别个人信息未经同意对外公开。
