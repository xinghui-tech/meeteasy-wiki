# VisuSpace 微站编辑

VisuSpace 是会易小蜜书的 **低代码微站编辑器**，组织者可为每场会议构建移动端友好的宣传与功能页面。

## 进入编辑器

1. Console → 选择会议 → **微站 / VisuSpace**。
2. 首次进入可 **从模板创建**（Admin 全局模板或租户私有模板）或 **空白页面**。

## 界面概览

| 区域 | 功能 |
|------|------|
| 左侧页面树 | 多页面管理、顺序调整 |
| 中央画布 | 所见即所得预览（移动端宽度） |
| 右侧属性栏 | 选中组件的样式与数据绑定 |
| 顶部工具栏 | 预览、保存、发布、AI 生成 |

## 常用组件

- **文本 / 标题**：品牌 slogan、会议介绍
- **图片 / 轮播**：主视觉、赞助商 Logo
- **按钮**：跳转报名、外链、锚点
- **议程列表**：绑定会议议程 API
- **嘉宾墙**：绑定嘉宾数据
- **AI 聊天框**：嵌入 MeetApp AI 助手入口

## 编辑流程

1. 在页面树 **添加页面**（如首页、议程、嘉宾、报名）。
2. 从组件面板拖入组件，在属性栏修改文案、颜色、间距。
3. 使用 **页面背景** 设置纯色、渐变或背景图。
4. **保存** 草稿 → **预览**（新窗口模拟 MeetApp）→ **发布**。

## AI 辅助生成

1. 点击 **AI 生成**，描述页面需求（如「科技峰会首页，橙色主色，含议程和报名按钮」）。
2. 系统基于 VisuSpace DSL 生成页面结构，可人工微调后保存。
3. DSL 规范见 [VisuSpace DSL](/developer/visuspace-dsl)。

::: tip 提示
AI 生成内容仅供参考，发布前请核对文案与品牌合规要求。
:::

## 发布与版本

- **发布** 后变更对 MeetApp 访问者可见。
- 建议在重大活动前锁定编辑权限，仅指定人员可发布。
- 历史版本回滚功能视产品版本而定。

## 与 MeetApp 的关系

MeetApp 通过会议 slug 加载已发布的 VisuSpace 页面路由；未发布页面仅编辑器内预览可见。
