# 报名配置与审核

Console 报名模块支持灵活的表单设计、审核流程与票码管理。

## 启用报名

1. 会议详情 → **报名设置**。
2. 打开 **启用在线报名**。
3. 设置报名时间窗口（开始/截止）。
4. 可选：总人数上限、是否允许重复报名（同一账号）。

## 表单设计（AMIS）

1. 进入 **表单设计**，基于 AMIS 低代码编辑器添加字段。
2. 常用字段：姓名、手机、邮箱、公司、职位、 dietary、发票信息。
3. 配置校验：必填、正则、唯一性（如手机号）。
4. 保存并 **发布表单** 后，MeetApp 报名页生效。

详见 [AMIS 表单](/user-manual/console/forms)。

## 审核流程

| 模式 | 说明 |
|------|------|
| 自动通过 | 提交即成功，立即生成票码 |
| 人工审核 | 组织者需在 Console 审批 |

### 人工审核操作

1. **报名列表** → 筛选 **待审核**。
2. 查看详情 → **通过** 或 **驳回**（可填驳回原因）。
3. 支持批量通过；通过后系统自动发送通知（如已配置邮件/模板消息）。

## 票码与参会状态

- 审核通过或自动通过后，系统生成唯一 **票码**（可用于签到扫码）。
- 列表可导出 CSV/Excel，含票码、字段、提交时间。
- 参会状态：`待审核` / `已通过` / `已驳回` / `已签到`。

## 参会者视角

参会者在 MeetApp 提交表单后的体验见 [报名与表单](/user-manual/meetapp/registration)。

## 注意事项

- 收集个人信息需在前端展示 [隐私政策](/legal/privacy) 链接并取得同意。
- 敏感字段（身份证等）请评估必要性，遵循最小必要原则。
- 报名截止后，MeetApp 入口自动关闭，Console 仍可手动添加参会者（如功能已启用）。
