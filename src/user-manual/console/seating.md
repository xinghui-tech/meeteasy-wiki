# 座位图与分配

Console 座位模块支持可视化座位图编辑、批量分配与 MeetApp 查询。

## 创建座位图

1. 会议详情 → **座位管理** → **新建座位图**。
2. 选择模板：剧院式、圆桌式、 classroom 等（或导入背景图）。
3. 在画布上添加 **区域**、**桌**、**座位**，设置编号规则（如 A 区 1 排 3 座）。

## 分配座位

### 手动分配

1. 在 **参会者列表** 中选择已通过审核的报名记录。
2. 点击 **分配座位** → 在座位图上点选空位。
3. 保存后，参会者可在 MeetApp 查看座位信息。

### 批量分配

- 按报名顺序自动填充空位。
- 按标签（VIP、媒体）分配到指定区域。
- 导入 CSV（票码/手机号 + 座位号）批量更新。

## 发布与查询

1. **发布座位图** 后，MeetApp **我的票券/签到页** 显示座位号。
2. 现场签到处扫码后可联动展示座位引导。
3. 未分配座位的参会者显示「待分配」或隐藏座位区块（可配置）。

## 现场变更

- 组织者可在 Console  **改座**（拖拽或重新选择）。
- 变更后 MeetApp 刷新即更新；建议重大调整通过广播通知参会者。

## 最佳实践

- 预留 5–10% 空位应对临时增员。
- VIP 区域单独分区，避免与大众票混排。
- 打印座位图前在 VisuSpace 添加「座位查询」入口链接。
