# 现场签到与座位查询

参会者使用 MeetApp 完成自助签到并查看座位信息。

## 自助签到

### 前置条件

- 报名已 **审核通过**
- 组织者已开启签到且在开放时间内

### 操作步骤

1. 打开 MeetApp → **我的票券** 或会议页 **签到** 入口。
2. 点击 **立即签到**。
3. 成功页显示签到时间与欢迎语；若已分配座位，同时展示 **座位号/桌号**。

## 扫码签到（签到处）

若签到处采用扫码模式：

1. 打开 **我的票券** 中的 **二维码**。
2. 工作人员扫描即可；您无需额外操作。
3. 屏幕显示「已签到」即完成。

## 查询座位

1. **我的票券** 详情区查看 **座位信息**。
2. 部分会议在微站提供 **座位查询** 页：输入姓名/手机查询（视组织者配置）。
3. 座位变更后刷新页面即可看到最新分配。

## 未分配座位

显示「座位待分配」时，请联系现场工作人员或在 Console 分配后再查看。

## 常见问题

**Q：提示「不在签到时间」？**  
A：签到尚未开放或已结束，请查看会议通知或咨询组织者。

**Q：已签到能否重复操作？**  
A：页面提示已签到，重复操作不影响统计。

**Q：代他人签到？**  
A：默认不支持；特殊情况由工作人员在 Console [手动签到](/user-manual/console/check-in)。
