# 报名与表单提交

参会者在 MeetApp 完成在线报名并查看审核状态的步骤说明。

## 进入报名页

1. 在会议微站点击 **立即报名** 或 **Register** 按钮。
2. 若会议未开放报名，按钮灰显或提示「报名已截止」。
3. 首次报名可能需要 **登录/验证** 手机号或微信。

## 填写表单

1. 按页面提示填写组织者配置的字段（姓名、手机、公司等）。
2. 必填项标有 * 号；格式错误时字段下方显示提示。
3. 提交前勾选 **同意隐私政策**（如展示）。
4. 点击 **提交**。

## 提交结果

| 状态 | 参会者看到的内容 |
|------|------------------|
| 自动通过 | 成功页 + 电子票码/二维码 |
| 待审核 | 提示「审核中」，请留意通知 |
| 驳回 | 显示驳回原因，可修改后重新提交（视配置） |

## 查看票券

1. 登录后进入 **我的** / **票券** 页。
2. 展示票码 QR、会议时间地点、座位号（如已分配）。
3. 现场出示 QR 供签到处扫描。

## 修改信息

- 报名截止前，部分会议允许在 **我的报名** 中编辑字段。
- 审核通过后修改关键信息（如姓名）可能需要联系组织者。
