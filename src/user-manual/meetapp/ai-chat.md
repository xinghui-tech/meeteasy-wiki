# AI 助手与语音输入

MeetApp 内置 AI 助手，帮助参会者快速获取会议相关信息；支持文字与语音输入。

## 打开 AI 助手

1. 在微站页面找到 **AI 聊天** 组件或浮动入口。
2. 首次使用需 **登录** 并同意 AI 服务相关说明。
3. 进入对话界面，可查看示例问题。

## 文字对话

1. 在输入框输入问题，例如：
   - 「主论坛几点开始？」
   - 「今天议程有哪些？」
   - 「会场在哪里？」
2. 点击发送，助手基于会议知识库与 LLM 生成回答。
3. 多轮对话保留上下文（同一会话内）。

## 语音输入

1. 点击输入框旁的 **麦克风** 图标。
2. 授权浏览器/微信 **麦克风权限**。
3. 说话结束后自动 ASR 转文字并发送（或需点击停止，视 UI 版本）。
4. 识别结果可编辑后再发送。

::: tip 提示
嘈杂环境可能影响识别准确度，建议在安静区域使用或改用文字输入。
:::

## 能力边界

- AI 回答 **仅供参考**，不构成法律、医疗等专业建议。
- 知识范围限于组织者提供的会议资料；未录入的内容可能无法准确回答。
- 请勿输入他人隐私或违法内容。见 [用户协议](/legal/user-agreement) AI 免责声明。

## 数据说明

对话内容可能用于改进服务质量，存储与 retention 策略见 [隐私政策](/legal/privacy)。您可申请删除相关会话数据（通过 [联系我们](/about/contact)）。

## 故障排除

| 问题 | 建议 |
|------|------|
| 无 AI 入口 | 确认组织者已启用 AI 插件且微站含聊天组件 |
| 回复慢 | 检查网络；高峰期限流属正常现象 |
| 麦克风不可用 | 检查系统权限；微信内需使用 HTTPS 页面 |
