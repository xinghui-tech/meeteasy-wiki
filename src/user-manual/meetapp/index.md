# 参会者入门（MeetApp）

**MeetApp** 是星汇小蜜书的参会者端 H5 应用，参会者通过链接或二维码访问会议微站、完成报名并使用 AI 助手与现场服务。

## 如何访问会议

1. 打开组织者分享的 **会议链接** 或 **扫描二维码**。
2. 链接格式通常为 `https://<域名>/meetapp/<tenant>/<conference-slug>/`（以实际部署为准）。
3. 微信小程序用户可通过 **WeApp** 打开同一 H5 页面。

## 浏览微站

- 首页及子页面由组织者通过 VisuSpace 发布，包含议程、嘉宾、合作伙伴等内容。
- 底部或页面内 **导航** 切换各子页。
- 支持深浅色主题（跟随系统或会议配置）。

## 登录与身份

| 场景 | 说明 |
|------|------|
| 仅浏览 | 多数公开页无需登录 |
| 报名 | 需手机号/邮箱验证或微信授权（视会议配置） |
| AI 助手 | 通常需登录以关联会话 |
| 签到 | 需已通过的报名记录 |

## 添加到主屏幕

移动端浏览器可 **添加到主屏幕**，获得接近 App 的全屏体验（iOS Safari / Android Chrome）。

## 帮助与反馈
