# 系统设置

Admin **系统设置**集中管理平台级参数，影响所有租户或作为各租户的默认基线。

## 设置分类

| 分类 | 典型配置项 |
|------|------------|
| 基础 | 站点名称、默认语言、时区 |
| 认证 | 会话超时、密码策略、登录方式 |
| AI | 默认 LLM 提供商、模型、Token 限额 |
| ASR | 语音识别服务地址与密钥 |
| 邮件/通知 | SMTP、短信网关（如已集成） |
| 存储 | 对象存储端点、CDN 域名 |

## AI 配置

1. 进入 **系统设置** → **AI**。
2. 选择提供商（如 DeepSeek、OpenAI 或自建兼容接口）。
3. 填写 API Base URL、API Key、默认模型 ID。
4. 设置单用户/单会议 Token 上限，防止滥用。
5. 保存后，在 [插件管理](/user-manual/admin/plugins) 中为租户启用 AI 插件。

::: warning 注意
AI 对话内容可能包含用户个人信息，处理规则见 [隐私政策](/legal/privacy)。
:::

## ASR 配置

用于 MeetApp 语音输入与转写场景：

1. 配置 ASR 服务 endpoint 与认证信息。
2. 选择默认识别语言（中文/英文等）。
3. 测试连接成功后保存。

## 认证与安全

- **会话超时**：建议生产环境 ≤ 24 小时，敏感操作可要求重新验证。
- **密码策略**：最小长度、复杂度、历史密码限制。
- **CORS / 域名白名单**：在部署层（Nginx）与后端 `.env` 中同步配置。

## 修改生效范围

- 标记为 **全局** 的配置立即影响新创建的会议与租户默认行为。
- 已创建会议的部分配置可能需组织者在 Console 中单独调整。
- 重大变更前建议在测试租户验证。
