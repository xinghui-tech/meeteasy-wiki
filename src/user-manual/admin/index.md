# 平台运营入门（Admin）

Admin 是星汇小蜜书的**平台管理端**，供星汇盛世（北京）科技有限公司及授权运营人员管理多租户、全局插件与系统配置。

## 登录与访问

1. 在浏览器打开 Admin 部署地址（由运维提供，通常为 `https://<域名>/admin/`）。
2. 使用平台管理员账号登录（手机号/邮箱 + 密码，或企业 SSO，视部署配置而定）。
3. 首次登录建议修改密码并确认双因素认证策略（如已启用）。

## 主要功能模块

| 模块 | 说明 |
|------|------|
| 租户管理 | 创建租户、分配配额、启用/停用 |
| 用户管理 | 平台级管理员与租户管理员账号 |
| 插件管理 | 安装、配置、按租户启用插件 |
| 系统设置 | AI 模型、ASR、邮件等全局参数 |
| 模板与素材 | 全局 VisuSpace 模板与共享素材库 |

## 租户管理

### 创建租户

1. 进入 **租户管理** → **新建租户**。
2. 填写租户名称、标识（slug）、联系人信息。
3. 配置初始配额（会议数量、存储上限等，视产品版本而定）。
4. 保存后，为该租户创建 **租户管理员** 账号，对方即可登录 Console。

### 停用租户

停用后，该租户下所有 Console / MeetApp 访问将受限，数据保留策略见 [数据安全](/legal/data-security)。操作前请与商务/法务确认。

## 用户管理

- **平台管理员**：可访问 Admin 全部功能。
- **租户管理员**：仅管理所属租户，在 Console 中操作会务。
- 支持按角色分配权限（RBAC），遵循最小权限原则。

## 常见操作

| 场景 | 操作路径 |
|------|----------|
| 新客户开通 | 租户管理 → 新建 → 创建租户管理员 |
| 启用 AI 插件 | 插件管理 → AI → 配置 API Key → 租户启用 |
| 全局模板下发 | 模板与素材 → 上传 → 租户可见范围 |
