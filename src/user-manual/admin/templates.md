# 全局模板与素材库

Admin 可维护 **VisuSpace 全局模板** 与 **共享素材**，供各租户 Console 在创建会议微站时复用，保证品牌一致性与制作效率。

## 全局模板

### 什么是全局模板

预设计的 VisuSpace 页面结构（DSL），包含常见会务场景布局，例如：

- 峰会首页（主视觉 + 议程摘要 + 报名按钮）
- 嘉宾介绍页
- 合作伙伴页
- 报名成功页

### 上传与管理

1. Admin → **模板与素材** → **全局模板**。
2. 点击 **上传模板**，选择 JSON/DSL 导出文件或从 VisuSpace 导出的包。
3. 填写模板名称、分类、预览图。
4. 设置 **可见租户**（全部 / 指定租户）。

### 租户侧使用

租户管理员在 Console → 会议 → VisuSpace 编辑器中，可从 **从模板创建** 选择 Admin 下发的模板，再按会议需求微调。

## 素材库

### 支持素材类型

- 图片（PNG、JPG、WebP、SVG）
- 品牌 Logo、背景图、图标
- 视频封面（链接型资源）

### 上传规范

| 项 | 建议 |
|----|------|
| 图片尺寸 | 首屏背景宽度 ≥ 750px（移动端基准） |
| 文件大小 | 单文件 ≤ 2MB（可按存储策略调整） |
| 命名 | 使用英文 slug，便于 DSL 引用 |

1. **素材库** → **上传** → 选择文件与分类标签。
2. 上传完成后获得 **资源 URL**，可在 VisuSpace 组件属性中粘贴或从素材选择器选取。

## 权限与审核

- 仅平台 Admin 可上传 **全局** 模板；租户可在 Console 保存 **私有** 模板（仅本租户可见）。
- 含第三方商标的素材需确保版权授权；违规内容按 [服务条款](/legal/terms) 处理。
