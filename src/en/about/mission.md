# Mission & Vision

## Our Mission

**Make conference management effortless for organizers and delightful for attendees.**

We believe every organization — from a 50-person workshop to a 5,000-person summit — deserves professional-grade event tools without enterprise complexity or cost. MeetEasy democratizes conference management by combining multi-tenant SaaS architecture, low-code customization, and AI-native attendee experiences.

## Our Vision

Within the next 3–5 years, we aim to:

- Become the **default meeting management platform** for organizations in China that value mobile-first, AI-enhanced event experiences
- Support **10,000+ conferences per year** across diverse industries and event formats
- Build an **open plugin ecosystem** where developers and partners extend the platform for specialized use cases
- Deliver **best-in-class attendee engagement** through AI assistants, real-time interaction, and seamless WeChat integration

## Core Values

Our values guide product decisions, engineering practices, and customer relationships:

### 1. Tenant First

Every feature, every query, every design decision starts with multi-tenant isolation. Our customers' data is sacred — we build trust through rigorous separation and security.

### 2. Open Extension

No two conferences are identical. Our plugin architecture ensures the platform adapts to customer needs rather than forcing customers to adapt to the platform.

### 3. Experience Above All

Organizers work on mobile during events. Attendees expect instant answers. We optimize every interaction for speed, clarity, and delight — not feature count.

### 4. Prefer Boring

We choose simple, testable, maintainable solutions over clever complexity. Reliability at event day is non-negotiable.

### 5. Bilingual by Default

China's conference landscape is bilingual. Every user-facing surface supports Chinese and English from day one — not as an afterthought.

## How Values Shape the Product

| Value | Product manifestation |
|-------|----------------------|
| Tenant First | Multi-tenant architecture, RBAC, data isolation |
| Open Extension | Plugin system (Embedded + Sidecar), VisuSpace DSL, AMIS forms |
| Experience Above All | Mobile-first Console/MeetApp, AI assistant, VisuSpace microsites |
| Prefer Boring | Monorepo, REST + Socket.IO, proven tech stack |
| Bilingual by Default | vue-i18n across all apps, zh-CN / en-US locale files |
