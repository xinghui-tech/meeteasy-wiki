# Team

MeetEasy is built by a team passionate about event technology, developer experience, and attendee engagement.

## Culture

We believe in:

- **Shipping with quality** — Reliable software for event day is our highest priority
- **Developer empathy** — Clean APIs, good documentation, and boring technology choices
- **Customer proximity** — We use our own platform to run internal events and gather feedback
- **Continuous learning** — AI, low-code, and mobile-first are evolving fast; we evolve with them

## Engineering Highlights

Our engineering team maintains:

- A **Python async backend** with FastAPI, Beanie, and FastStream
- A **Vue 3 monorepo** with six frontend applications sharing a WebAPI SDK
- A **VisuSpace DSL** for declarative microsite page generation
- A **plugin system** supporting Embedded Python and Sidecar HTTP extensions
- **Bilingual i18n** across all user-facing surfaces

See the [Developer Guide](/en/developer/architecture) for technical details.
