# About MeetEasy

**MeetEasy** (星汇小蜜书) is an enterprise meeting management SaaS platform developed by **Xinghui Shengshi (Beijing) Technology Co., Ltd.** We help organizations of all sizes plan, execute, and measure conferences — from registration to on-site engagement and post-event analytics.

## The Problem

Conference organization is fragmented. Organizers juggle separate tools for registration, agenda management, attendee communication, check-in, seating, and analytics. Attendees face inconsistent experiences across email links, spreadsheets, and ad-hoc apps. Platform operators struggle to serve multiple organizations without robust multi-tenancy.

## Our Solution

MeetEasy unifies the entire conference lifecycle in one multi-tenant platform:

- **For organizers** — Create events, design registration forms, build microsites, manage check-in and seating, and track analytics — all from a mobile-first Console.
- **For attendees** — Browse conference microsites, register, chat with an AI assistant, check in, and find seats — through MeetApp or WeChat.
- **For platform operators** — Manage tenants, configure plugins, and maintain global templates through Admin.

## Core Capabilities

| Capability | Description |
|------------|-------------|
| Multi-tenant SaaS | Strict data isolation for multiple organizations on one platform |
| Full lifecycle | Registration → check-in → seating → analytics in one system |
| Low-code tools | AMIS forms and VisuSpace DSL for rapid customization |
| AI-native | Built-in LLM assistant and voice input for attendees |
| Plugin extensibility | Embedded and Sidecar plugins for custom integrations |
| WeChat integration | Native mini programs with seamless SSO |

## Who We Serve

- **Corporate event teams** running internal summits, training, and all-hands meetings
- **Industry associations** managing annual conferences and member events
- **Event agencies** serving multiple clients on a shared platform
- **SaaS platform operators** offering meeting management as a white-label service
