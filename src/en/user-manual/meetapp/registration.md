# Registration & Forms

Complete conference registration and submit forms as an attendee in MeetApp.

## Register for a Conference

1. Open the conference in MeetApp.
2. Tap **Register** on the homepage or navigation menu.
3. Fill in the registration form with required information.
4. Submit the form.

## After Submission

What happens next depends on the organizer's settings:

| Approval mode | Your experience |
|---------------|-----------------|
| **Auto-approve** | Immediate confirmation with ticket code |
| **Manual review** | Pending status until organizer approves |

You'll see your registration status on the conference page:

- **Pending** — Awaiting organizer review
- **Approved** — Registration confirmed; ticket code displayed
- **Rejected** — Registration declined (reason may be shown)

## Ticket Code

If ticket codes are enabled:

1. After approval, your unique ticket code appears on the registration confirmation page.
2. Save or screenshot the code for check-in.
3. You can also find the code in your profile or registration details.

## Submit Additional Forms

Organizers may publish standalone forms beyond registration:

- Post-event surveys
- Session feedback
- Custom data collection

Access these forms via links on the microsite or direct URLs shared by the organizer.

## Edit Registration

Whether you can edit after submission depends on organizer settings:

- If editing is allowed, tap **Edit Registration** on your confirmation page
- If not, contact the organizer to request changes

## Tips

- Complete registration early — popular events may reach capacity
- Double-check email and phone for confirmation notifications
- Keep your ticket code accessible for fast check-in on event day
- Fill optional fields that help the organizer (dietary needs, accessibility requirements)

See [Check-in](./check-in) for using your ticket code on event day.
