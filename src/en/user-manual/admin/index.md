# Admin — Getting Started

The **Admin** portal is MeetEasy's platform operations console. Platform operators use it to manage tenants, configure plugins, adjust system settings, and maintain global templates.

## Prerequisites

- A super-admin or platform operator account assigned by MeetEasy
- Desktop browser (Chrome, Firefox, Safari, or Edge recommended)
- Admin portal URL: https://m-admin.xinghui.club

## Sign In

1. Open the Admin portal in your browser.
2. Enter your administrator credentials (email/phone and password).
3. Complete any required two-factor verification.

## Tenant Management

Tenants represent organizations (customers) on the platform.

### Create a Tenant

1. Navigate to **Tenants** → **Create**.
2. Fill in organization name, contact info, and subscription plan.
3. Assign an initial tenant admin user.
4. Save and notify the tenant admin of their credentials.

### Manage Existing Tenants

- **Edit** — Update organization details and contact information
- **Suspend** — Temporarily disable tenant access
- **Configure** — Set tenant-scoped plugin and AI settings

## User Management

Platform-level user administration:

- Create and deactivate platform operator accounts
- Reset passwords and manage roles
- View login history and audit logs

Tenant-scoped user management is handled by tenant admins within Console.
