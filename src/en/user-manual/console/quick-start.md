# Quick Start — Create and Publish a Conference

This guide walks conference organizers through creating and publishing their first event in **Console**.

## Prerequisites

- Organizer account with access to your organization's tenant
- Console URL or WeConsole mini program
- Basic event details ready: name, dates, venue

## Step 1: Sign In

1. Open Console in your mobile browser, or launch **WeConsole** in WeChat.
2. Log in with your organizer credentials.
3. You will see your conference list (empty if this is your first event).

## Step 2: Create a Conference

1. Tap **Create Conference** (+).
2. Enter basic information:
   - **Conference name** — e.g., "2026 Annual Summit"
   - **Dates** — Start and end date/time
   - **Venue** — Location or "Online"
   - **Description** — Brief summary for attendees
3. Optionally import from a [global template](/en/user-manual/admin/templates).
4. Save as draft.

## Step 4: Build the Microsite

1. Open **VisuSpace** from the conference management menu.
2. Create a homepage with conference name, countdown, and registration button.
3. Add pages for agenda preview, speaker list, and venue map.
4. Preview on mobile and publish pages.

See [VisuSpace Microsite](./visuspace.md) for detailed editing instructions.

## Step 5: Configure Registration

1. Go to **Registration** → **Form Design**.
2. Use the AMIS form builder to add fields (name, email, company, custom fields).
3. Set approval mode: auto-approve or manual review.
4. Enable ticket code generation if needed.

See [Registration Configuration](./registration.md) for details.

## Step 6: Publish

1. Review all settings in the conference summary.
2. Tap **Publish** to make the conference live.
3. Share the MeetApp link or QR code with potential attendees.
