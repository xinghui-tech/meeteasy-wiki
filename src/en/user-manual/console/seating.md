# Seating Management

Design seat maps and assign seats to registered attendees for your conference.

## Overview

The seating module lets organizers:

- Create visual seat maps for venues (theaters, banquet halls, classrooms)
- Assign seats to individual registrants manually or automatically
- Display seat information to attendees after check-in

## Create a Seat Map

1. Go to **Conference** → **Seating** → **Seat Map**.
2. Choose a layout template or start from blank:
   - **Theater** — Rows and numbered seats
   - **Banquet** — Round tables with seat positions
   - **Custom grid** — Flexible row/column configuration
3. Define sections, rows, and seat numbers.
4. Mark seats as available, reserved, or blocked (aisles, equipment).

## Assign Seats

### Manual Assignment

1. Open **Seating** → **Assignments**.
2. Select a registrant from the list.
3. Click a seat on the map to assign.
4. Save assignment; attendee is notified.

### Bulk Assignment

1. Select multiple registrants (by registration order, ticket type, etc.).
2. Choose **Auto-assign** to fill available seats sequentially.
3. Review and adjust individual assignments as needed.

## Attendee Seat Lookup

After assignment, attendees can view their seat in MeetApp:

- **Before check-in** — Seat shown on registration confirmation (if enabled)
- **After check-in** — Seat highlighted on the venue map

See [MeetApp Check-in](/en/user-manual/meetapp/check-in) for the attendee experience.

## Reserved & VIP Seats

- Mark specific seats as **VIP** or **Reserved** before opening general assignment
- Assign VIP seats to speakers, sponsors, or special guests manually
- Block seats near exits, stages, or AV equipment

## Export

- Export seat assignment list with attendee names and seat numbers
- Generate printable seat chart for venue staff
- Include seat info in check-in confirmation messages

## Tips

- Finalize seat map layout before bulk assignment to avoid rework
- Leave buffer seats near aisles for accessibility
- Coordinate with check-in: enable seat display at check-in for smoothest on-site flow
- For large events, assign seats in batches by ticket type or registration date
