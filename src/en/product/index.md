# Product Overview

**MeetEasy** (会易小蜜书) is an enterprise-grade, multi-tenant meeting management SaaS platform. It helps organizations plan, run, and measure conferences of any scale — from corporate summits and industry forums to training sessions and hybrid events.

## What MeetEasy Does

MeetEasy covers the full conference lifecycle in one platform:

- **Before the event** — Create conferences, design registration forms, build VisuSpace microsites, and configure agendas, speakers, and seating.
- **During the event** — Enable attendee check-in, on-site seat lookup, AI assistant support, and real-time notifications.
- **After the event** — Review analytics, share attribution, registration summaries, and export data for follow-up.

## Who It Is For

| Role | Primary application | Typical tasks |
|------|---------------------|---------------|
| Platform operator | Admin | Tenant management, global plugins, system settings, templates |
| Conference organizer | Console / WeConsole | Create events, manage registration, check-in, seating, forms, analytics |
| Attendee | MeetApp / WeApp | Browse microsite, register, use AI chat, check in, find seats |
| Page designer | VisuSpace | Build and publish conference microsite pages with the visual editor |
| Developer | API & plugins | Extend the platform with Embedded or Sidecar plugins |

## Core Value Proposition

1. **Multi-tenant by design** — One platform serves many organizations with strict data isolation.
2. **Low-code where it matters** — AMIS forms and VisuSpace DSL reduce custom development for registration and microsites.
3. **AI-native attendee experience** — LLM chat and voice input (ASR) are built in, not bolted on.
4. **Open extensibility** — Plugin architecture supports AI providers, OCR, and custom business logic.
5. **Unified analytics** — PV/UV, share tracking, and daily rollups give organizers actionable insight.
