# VisuSpace AI 低代码微站编辑器

VisuSpace 是星汇小蜜书的核心 **AI 辅助低代码页面设计器**。它允许会议组织者（B端客户）通过可视化拖拽或自然语言描述（AI 提示词），快速为每场会议搭建移动端友好的宣传主页、议程展示、嘉宾墙、报名通道和现场签到微站。

---

## 1. 核心设计理念

1. **AI 驱动开发与生成**：
   整个 VisuSpace 渲染引擎及页面架构由 AI 协同开发和自主维护。用户可以通过自然语言向 AI 提出需求，系统会自动解析并生成合法的 JSON DSL，并在画布中渲染呈现。
2. **数据源原生绑定**：
   低代码组件（如议程、嘉宾、座位、签到）可直接与 MeetEasy 会议数据库（FastAPI 后端）进行数据联动，无需手动编写 API 接口调用代码。
3. **一次配置，多端渲染**：
   设计出的微站自动适配 MeetApp（移动端 H5）、微信小程序（WeApp 内嵌 WebView）等多种参会终端。
4. **统一主题与样式变量**：
   支持全局主题色、圆角、背景渐变，自动适配深浅色模式，保证品牌视觉的高度一致。

---

## 2. 编辑器界面概览

VisuSpace 编辑器由以下四个核心区域组成：

| 区域 | 功能说明 |
| :--- | :--- |
| **左侧页面树 (Page Tree)** | 管理微站的多页面（如首页、日程页、嘉宾页、地图引导页）。支持页面增删、自定义路由命名（Slug）、图标选择及顺序拖拽。 |
| **中央画布 (Canvas)** | **所见即所得 (WYSIWYG)** 预览区域，采用 375px 移动端基准宽度，模拟真实的手机端渲染效果。 |
| **右侧属性栏 (Properties Panel)** | 选中具体组件或页面背景时，在此修改具体业务属性（如文本内容、图片链接、跳转动作）与外观样式（如内边距、外边距、背景色）。 |
| **顶部工具栏 (Toolbar)** | 提供“保存草稿”、“新窗口预览”（模拟真实 MeetApp 容器环境）、“发布上线”以及关键的 **“AI 生成”** 功能。 |

---

## 3. AI 辅助与工具生态支持

* **自然语言生成**：通过右侧或顶部的 AI 对话框，输入如 *“生成一个暗黑风格的峰会首页，包含开幕倒计时和议程组件”*，AI 即可根据上下文库快速编写出符合 Schema 要求的 DSL，零代码门槛实现冷启动。
* **RAG 知识整合**：当组织者与 AI 交互以填充微站内容时，系统内置的 **RAG（检索增强生成）** 服务会自动检索会议自身的 FAQ、简介、嘉宾背景等结构化数据，确保 AI 填充的文案准确合规。
* **MCP（Model Context Protocol）**：支持 Model Context Protocol 协议。外部的 AI 编程 Agent 或自主 Agent 能够直接通过标准工具读取微站 DSL，实现页面自动化更新与数据同步。
