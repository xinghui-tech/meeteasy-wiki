## ADDED Requirements

### Requirement: Step-by-step conference assistant tutorial
The system SHALL provide a step-by-step tutorial explaining how to build a complete conference assistant (micro-site) from scratch, covering initial creation, adding features, previewing, and final publishing.

#### Scenario: User follows the tutorial to build a micro-site
- **WHEN** a new conference organizer reads the "0 to 1" guide
- **THEN** they SHALL find an end-to-end tutorial detailing how to set up pages for agenda, registration, map navigation, and live check-in

### Requirement: AI assistant prompt generation instructions
The tutorial SHALL explain how to leverage the AI generation feature in VisuSpace to output draft page configurations, including recommended prompts and tips for refinement.

#### Scenario: Organizer uses AI to scaffold a page
- **WHEN** an organizer wants to create a page using natural language instructions
- **THEN** they SHALL find detailed guidance on how to input prompts (such as "科技峰会首页，橙色主色...") and how to fine-tune the resulting layouts
